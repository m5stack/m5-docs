M5GO RGB
========

DESCRIPTION
-----------

The M5GO RGB is a sensor including three RGB LED lights and two GROVE
ports.You can display specified color using M5GO Core through Blockly,
Arduino or MicroPython.

FEATURES
--------

-  Two Lego installation holes
