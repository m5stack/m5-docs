## 描述

**HUB** 是一款 GROVE 接口的拓展器,其功能是能将 1个 GROVE 接口拓展为3个，使得能够同时连接更多的Unit传感器.

**注意:**
同时连接 2 - 3 个 IIC 接口 (红色接口) 同样的Unit 的话,将会引起 IIC 地址冲突.使用 IIC HUB Unit 取代 HUB Unit 能够以避免 IIC 地址冲突的发生.

### 产品描述

- GROVE 拓展器
- 2x LEGO 兼容孔

## 包含

- 1x HUB Unit
- 1x Grove 线
